## Tentang Aplikasi

Front Office App adalah aplikasi manajemen front office yang dikembangkan untuk membantu pengelolaan pengunjung dan karyawan. Aplikasi ini dibangun menggunakan PHP, MySQL, JavaScript, dan CSS.
